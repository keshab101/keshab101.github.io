Phonebook App

Simple hash table project written in Java

This was a class project for my Advanced Data Structure class at my university. The project was fully implemented on Java. Although it does not include the GUI to make the project interactable, it does include the entire logic and full implementation with classes and methods.

In this project, I have implemented an abstraction over phonebook; a collections of pairs of type [Full name, Phone Number]. The phonebook supports both name-based search and phone-based search. For efficiency, the phonebook internally maintains a pair of hash tables from String s to String s, one with person's name as a key and phone number as a value. Entry uniqueness is guaranteed.

  • Date Implemented: Summer 2020
  • Class: Advanced Data Structures

Spatial

A KD-Tree and PR-Quad Trees Project in Java

Similar to the previous project, this was a project for my Advanced Data Structures class. This project was fully implemented on Java using Eclipse. The project does not include the Graphical User Interface (GUI) for Java but it does include all the source code with written tests to test the project.

This project is an implementation of KD-Trees and P-R (Point Region) QuadTrees. I have implemented both dictinary and spatial query functionality in this project. This project is heavily implemented with Object-Oriented Programming primitives, in particular, Inheritance and Polymorphism
